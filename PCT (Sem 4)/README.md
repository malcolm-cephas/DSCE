# Principles of Communication Theory (Sem 4) - Lab Programs

This directory contains MATLAB simulation programs extracted from the Integrated Principles of Communication Theory Lab.

## List of Experiments

| Experiment No. | Experiment Title | Filename |
| :--- | :--- | :--- |
| [**Expt 1**](./Expt1_RandomVariable.m) | Random Variables (Marks PDF, CDF, Statistics) | `Expt1_RandomVariable.m` |
| [**Expt 2**](./Expt2_NoiseGeneration.m) | Thermal and Shot Noise Generation | `Expt2_NoiseGeneration.m` |
| [**Expt 6**](./Expt6_PCMSimulation.m) | Pulse Code Modulation (PCM) Simulation | `Expt6_PCMSimulation.m` |
| [**Expt 9**](./Expt9_FMSimulation.m) | Frequency Modulation (FM) and Demodulation | `Expt9_FMSimulation.m` |
| [**Expt 10**](./Expt10_AMSimulation.m) | Amplitude Modulation (AM) and Demodulation | `Expt10_AMSimulation.m` |
| [**Expt 10 (Ext)**](./Expt10_AMDifferentIndices.m) | AM for Different Modulation Indices | `Expt10_AMDifferentIndices.m` |

## How to Run
1. Open **MATLAB**.
2. Set the current folder to this directory.
3. Open any of the `.m` files.
4. Press **Run** or type the filename in the Command Window.

## Note
These programs represent the simulation (Part A) components of the laboratory curriculum. Hardware-related experiments (Part B) involving kits like the Link-B or 555 timers are described in the manual but do not have software source code associated with them.
